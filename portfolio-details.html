Calculadora de Métricas Digitais

Este projeto foi desenvolvido para auxiliar profissionais de marketing digital a calcularem métricas importantes de forma rápida e precisa. Inclui calculadoras para:

  • ROI (Return on Investment)
  • CPC (Custo por Clique)
  • CPM (Custo por Mil Impressões)
  • CTR (Click-Through Rate)
  • Taxa de Conversão
  • E outras métricas essenciais

Desenvolvido com HTML, CSS e JavaScript, o projeto possui interface intuitiva e responsiva, facilitando o acesso em qualquer dispositivo.
